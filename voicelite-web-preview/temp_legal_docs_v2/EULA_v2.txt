5. DATA COLLECTION AND PRIVACY

a) VOICE RECORDINGS:
   - Audio recordings are processed locally on your device
   - Recordings are NEVER uploaded to VoiceLite servers
   - Audio files are automatically deleted after transcription
   - No third-party services process your voice data

b) SUBSCRIPTION DATA (Pro Tier Only):
   - Email address (for account management and license delivery)
   - Payment information (processed securely via Stripe, not stored by VoiceLite)
   - License activation data (device count, activation timestamps)

c) DEVICE FINGERPRINTING (Pro Tier Only):
   - Device fingerprints are generated using: CPU ID + Machine GUID (both values hashed using SHA-256)
   - Used solely for license activation and device limit enforcement (3 devices per Pro license)
   - Fingerprints are stored in encrypted form on VoiceLite servers
   - Cannot be reverse-engineered to identify individual users or machines

d) ANALYTICS DATA (Opt-In Only):
   - VoiceLite includes privacy-first analytics that require explicit user consent
   - You will see a consent dialog on first launch - analytics are DISABLED by default
   - Anonymous User ID: SHA-256 hash of (Machine ID + Timestamp) - irreversible, no PII
   - Events tracked (if you opt in):
     * APP_LAUNCHED: App starts (tier, version, OS version)
     * TRANSCRIPTION_COMPLETED: Aggregated daily counts (total words, model used, no content)
     * MODEL_CHANGED: Model switches (old model â†’ new model)
     * SETTINGS_CHANGED: Setting name only (no values)
     * ERROR_OCCURRED: Error types for debugging (no stack traces, no file paths)
     * PRO_UPGRADE: Pro tier activation events
   - What is NOT collected:
     * No recording content or transcription text
     * No file paths or directory structures
     * No IP addresses (backend does not log IPs)
     * No personally identifiable information (PII)
     * No sensitive user data
   - You can disable analytics anytime in Settings
   - Analytics failures never break app functionality (silent failures)
   - Data retention: 2 years, then automatically deleted

e) CRASH REPORTS:
   - Error logs are stored locally in %LOCALAPPDATA%\VoiceLite\logs\
   - Logs are NEVER automatically uploaded
   - You may voluntarily submit logs for support via contact@voicelite.app

For full privacy details, see our Privacy Policy at: https://voicelite.app/privacy
